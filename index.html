Se vocês hospedarem este arquivo no Netlify ou GitHub Pages, podem conectá-lo ao próprio servidor do Google (Firebase) de forma **totalmente gratuita** para ter sincronização automática entre dispositivos.

Como Ativar:

  1. Crie uma conta em firebase.google.com.
  2. Crie um novo projeto chamado "Sintonia Financeira".
  3. Ative a opção **Firestore Database** e selecione o modo de teste ou regras públicas.
  4. Ative o **Anonymous Authentication** na aba Auth.
  5. Adicione um app da Web nas configurações e cole as chaves do seu projeto abaixo.
